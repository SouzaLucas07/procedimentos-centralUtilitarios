## 🛠️ Procedimentos - Central de Utilitários

O Procedimentos é um portal web desenvolvido em React projetado para reunir diversas ferramentas utilitárias e pequenos sistemas em um único lugar. O objetivo é oferecer uma interface intuitiva para tarefas rápidas do dia a dia, como cálculos, localização de endereços e organização pessoal.

## 🚀 Funcionalidades Atuais

🔢 Calculadora: Interface para operações matemáticas rápidas.

📍 Localizador de Endereços: Busca de endereços via CEP de forma automatizada.

🖱️ Contador de Clicks: Ferramenta para monitoramento de frequências e interações.

🎮 Jogo da Velha: Implementação do clássico jogo de estratégia para lazer.

✅ To-do List: Gerenciador de tarefas para organização e produtividade.

💻 Tecnologias Utilizadas

## O projeto foi construído utilizando as seguintes tecnologias:

React: Biblioteca principal para a construção da interface.

React Router Dom: Gerenciamento de rotas e navegação entre as ferramentas.

JavaScript (ES6+): Lógica de programação e manipulação de dados.

CSS3: Estilização responsiva e componentes visuais.

## 🛠️ Como Executar o Projeto
Clone o repositório:
```
git clone https://github.com/seu-usuario/procedimentos.git
```
## Instale as dependências:
```
npm install
```
## Inicie o servidor de desenvolvimento:
```
npm start
```

## Acesse no navegador:
```
http://localhost:3000
```

## 📂 Estrutura de Pastas

A organização do código segue o padrão de componentes e páginas para facilitar a manutenção:

```
src/
├── components/   # Componentes reutilizáveis (Header, Cards)
├── pages/        # Telas individuais de cada ferramenta
├── styles/       # Arquivos de estilização CSS
└── App.js        # Estrutura principal e roteamento
```
